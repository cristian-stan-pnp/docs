## Moneda[​](#moneda "Enlace directo al Moneda")

La moneda soportada para este servicio son los pesos mexicanos.

## Tarjetas para entorno de pruebas[​](#tarjetas-para-entorno-de-pruebas "Enlace directo al Tarjetas para entorno de pruebas")

Para realizar operaciones con y sin 3DS, se deben proporcionar los siguientes datos:

```
- número de tarjeta: 4111 1111 1111 1111- CVV: 123- fecha de expiración: 12-34
```

Tras introducir los datos de tarjeta, el proceso solicitará el código de seguridad y deberemos introducir el valor 555. Ambos flujos deberían ser exitosos.

## Códigos de error[​](#códigos-de-error "Enlace directo al Códigos de error")

La estructura del mensaje de error puede tener los siguientes formatos:

```
Codigo: Mensaje
```

| Código | Mensaje                                                                                                    |
| ------ | ---------------------------------------------------------------------------------------------------------- |
| 01     | El documento XML no cumple con el esquema definido                                                         |
| 02     | El documento XML no cumple con las restricciones                                                           |
| 03     | Los datos de empresa, sucursal o usuario son incorrectos                                                   |
| 04     | Número de tarjeta inválido                                                                                 |
| 05     | Múltiples errores con la configuración de la sucursal                                                      |
| 06     | Error de la validación de la tarjeta (VISA, MASTERCARD o AMEX)                                             |
| 07     | No se puede cobrar a meses con este banco, o el banco no concuerda con el banco de la tarjeta              |
| 08     | El monto mínimo proporcionado es menor que el monto definido para la afiliación                            |
| 09     | El monto proporcionado no puede ser cobrado por este usuario                                              |
| 10     | La fecha de transacción está fuera de fecha de la afiliación requerida                                      |
| 11     | El monto proporcionado no puede ser cobrado por esta afiliación, motivo de transacción aprobada o rechazada |
| 12     | La tarjeta no puede ser procesada con esta afiliación                                                      |
| 13     | Código de seguridad CVV incorrecto                                                                         |
| 14     | Referencia inválida, esta operación solo puede ser realizada por el usuario el mismo dia                   |
| 15     | Los datos de la empresa son incorrectos                                                                    |
| 16     | Los datos del importe son incorrectos                                                                      |
| 17     | El número de autorización no coincide, importe 0.0 no válido                                               |
| 18     | La transacción original no fue aprobada                                                                     |
| 19     | American Express no puede realizar devoluciones                                                            |
| 20     | La operación ya ha sido realizada el día dd / mm / yyyy                                                    |
| 21     | Código de usuario incorrecto                                                                               |
| 22     | No tiene permisos para realizar esa operación                                                              |
