## Introducción

Apolopag es un servicio que permite realizar pagos por tarjeta o mediante los métodos de pago alternativos PIX y PicPay. También permite PIX Payouts. Puede ser utilizado tanto en el checkout de Paylands como mediante redirección, así como con pagos push.

## Datos extra requeridos

Apolopag espera recibir la siguiente información obligatoriamente en el campo `extra_data` al crear la orden, para pagos con tarjeta o para pagos PUSH con PIX y PicPay:

* Nombre y apellidos
* Email
* Documento de identificación (CPF)
* Teléfono
* Fecha de nacimiento (opcional para pagos con tarjeta)

Estos campos pueden aportarse con la siguiente estructura en `extra_data`:

```
"extra_data": {    "profile": {        "first_name": "John",        "last_name": "Doe",        "email": "johndoe@paynopain.com",        "document_identification_number": "659.516.937-13",        "birthdate": "1996-01-01",        "phone": {            "number": "5151428089"        }     }}
```

## Payouts

Para usar PIX Payouts con Apolopag es necesario enviar la siguiente información en el `extra_data`:

* Nombre y apellidos
* Documento de identificación (CPF)
* Teléfono
* Código de banco
* Número de agencia bancaria
* Número de cuenta
* Tipo de cuenta (current/saving/salary/payment)

```
"extra_data": {    "profile": {        "first_name": "John",        "last_name": "Doe",        "document_identification_number": "842.359.516-10"     },    "payment": {        "bank_code": "218",        "branch": "0944",        "account": "091234-5",        "account_type": "current"    }}
```

## Métodos de pago en Sandbox

La tarjeta de prueba es la siguiente. En general, la fecha de caducidad y CVV no importan siempre y cuando la fecha sea posterior a la fecha actual, y el CVV tenga 3 cifras.

```
- número de tarjeta: 4111 1111 1111 1111- CVV: 123- fecha de expiración: 12/34
```

## Moneda

La moneda soportada para este servicio es el real brasileño (BRL).

## Códigos de error

La estructura del error tiene la siguiente forma:

```
Código - Mensaje
```

Por ejemplo:

```
400 - first and last name are required
```
