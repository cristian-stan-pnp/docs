Se admiten dos métodos de pago: Tarjeta y [Bizum](/docs/apm/apm-bizum). Las operaciones de devolución, confirmación, cancelación y recuperación de estado son compatibles con ambos métodos de pago. El resto de operaciones están orientadas solamente a tarjetas.

## Restricciones y requisitos

Redsys soporta al menos las siguientes monedas: AUD, BGN, CAD, CHF, COP, CZK, DKK, EUR, GBP, HRK, HUF, ILS, INR, JPY, MYR, NOK, NZD, PEN, PLN, RUB, SAR, SEK, SGD, THB, USD, ZAR.

### Extra data

No se requiere indicar `extra_data` al crear la orden para llevar a cabo los pagos.

### Dynamic descriptor

El campo `dynamic_descriptor` es opcional y en caso de indicarse solo podrá contener un máximo de 38 caracteres y permite números, letras y espacios en blanco.

### Reference

El campo `reference` es opcional. En caso de indicarse, este valor aparecerá en **Canales** de Redsys dentro del campo `descripcion`, junto con el identificador `uuid` de la transacción.

El formato mostrado en Canales será:

```
uuid_transaccion/reference
```

⚠️ Importante: el comercio **no debe incluir** el `uuid` ni la barra `/` en el valor de `reference`. Estos se añaden automáticamente. El `uuid` corresponde a la transacción generada en nuestro sistema.

Por ejemplo, si el comercio envía como `reference`:

```
1234
```

En Canales se mostrará:

```
10FF94B5-FE7C-41A0-B956-BBBBAA6BCA77/1234
```

Este campo permite a los comercios añadir información propia para facilitar la trazabilidad de las operaciones. El tamaño máximo de este campo es **88 caracteres**. Si se supera este límite, el valor de `reference` será **truncado** en Canales.

## Métodos de pago en Sandbox

Las tarjetas de prueba de Redsys son las siguientes. En general, la fecha de caducidad y CVV no importan siempre y cuando la fecha sea posterior a la fecha actual, y el CVV tenga 3 cifras.

| Resultado                | Tarjeta                    |
| ------------------------ | -------------------------- |
| Pago OK                  | 4548812049400004 12/34 123 |
| Pago KO                  | 4111111111111111 12/34 123 |
| Soft-decline (No seguro) | 4548817212493017 12/34 123 |
