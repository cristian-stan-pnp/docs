---
title: 'Pagos con PROXY PCI'
description: 'Cómo procesar pagos a través del PROXY PCI de Paylands'
---

Para emitir un cargo a una tarjeta guardada, primero tiene que generar una orden de pago. Esto se hace utilizando [el método de generación de órdenes](https://docs.paylands.com/reference/#operation/generate-payment-order). Tiene que utilizar el token de la tarjeta obtenido durante el proceso de tokenización y enviarlo como valor del campo `source_uuid`. Después de esto, puede [redirigir al usuario a un formulario de pago](https://docs.paylands.com/reference/#operation/redirection-payment) o [utilizar el servicio web directamente](https://docs.paylands.com/reference/#operation/webservice-payment).

<Info>
  Por favor, consulte la documentación completa del PAYMENT GATEWAY: [https://docs.paylands.com](https://docs.paylands.com)
</Info>

Para más información y precios, póngase en contacto con [soporte@paylands.com](mailto:soporte@paylands.com).
