# Checkout de Paylands

El **Checkout de Paylands** es una solución flexible y robusta para aceptar pagos online de forma segura. Permite al usuario introducir sus datos de pago en un entorno alojado y seguro, compatible con múltiples métodos de pago.

## Modos de integración

Puedes integrar el Checkout de Paylands de tres formas distintas, según las necesidades técnicas de tu proyecto:

<CardGroup cols={3}>
  <Card title="Integración via API" icon="server" href="/guides/checkout/api-integration">
    Genera una orden desde tu servidor y redirige al cliente a la URL del checkout. El método más sencillo.
  </Card>
  <Card title="JavaScript SDK" icon="js" href="/guides/checkout/sdk-integration">
    Incorpora el checkout directamente en tu web usando nuestra librería JavaScript sin salir de tu sitio.
  </Card>
  <Card title="React Component" icon="react" href="/guides/checkout/react-integration">
    Usa el componente oficial de React para una integración nativa en proyectos basados en React/Next.js.
  </Card>
</CardGroup>

## Características

- **Múltiples métodos de pago**: Tarjeta, Google Pay, Apple Pay, Bizum, PayPal, Revolut Pay, y más.
- **3D Secure**: Autenticación fuerte del titular conforme a PSD2.
- **Personalizable**: Adapta el look & feel al branding de tu negocio.
- **Modo oscuro y claro**: Seleccionable mediante configuración.
- **Responsive**: Diseño adaptado a móviles y escritorio.
- **Iframe y redirección**: Compatible con ambos modos de visualización.

## Requisitos previos

Antes de integrar el Checkout asegúrate de:

1. Tener una cuenta activa en Paylands (Sandbox o Producción).
2. Obtener tu **API Key** desde el [Portal Paylands → Desarrolladores](https://portal.paynopain.com/settings/developers).
3. Haber configurado al menos un **Servicio de pago** en tu cuenta.
4. Haber creado al menos un **Checkout** desde el panel de Paylands.

<Tip>
  Si aún no tienes una cuenta, puedes registrarte en [portal.paynopain.com/register](https://portal.paynopain.com/register) o contactar con nuestro equipo de soporte.
</Tip>
