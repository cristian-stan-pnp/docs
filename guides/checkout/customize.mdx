# Personalizar un Checkout

Paylands permite personalizar el aspecto visual del checkout para adaptarlo al branding de tu negocio. Accede a la pestaña **"Personalización"** dentro de la configuración de tu checkout en el panel de administración.

## Opciones de personalización

| Elemento | Opciones / Descripción |
|----------|----------------------|
| **Título** | Texto que se muestra en la cabecera del checkout. |
| **Tema** | `Claro` (Light) o `Oscuro` (Dark). |
| **Color principal** | Color de los botones y elementos de acento. Formato HEX (p. ej. `#ee1b4e`). |
| **Fuente** | Tipografía del checkout. Soporta fuentes de Google Fonts. |
| **Logo** | Imagen del logotipo de tu marca (PNG, SVG). Se muestra en la cabecera. |
| **URL de términos y condiciones** | Enlace a los términos y condiciones que acepta el usuario antes de pagar. |
| **Campos adicionales** | Campos extra del formulario (p. ej. teléfono, comentario). |

## Buenas prácticas

- Usa el **color corporativo** de tu marca como color principal para reforzar la confianza del usuario.
- Proporciona siempre una **URL de términos y condiciones** válida; es un requisito legal en muchos países.
- Si tu web usa modo oscuro, elige el tema oscuro en el checkout para lograr coherencia visual.
- El logo se mostrará en la parte superior del formulario de pago. Usa una imagen con fondo transparente para mejor integración.

## Vista previa

Puedes ver una previsualización del checkout en tiempo real desde el panel de administración mientras editas la configuración de personalización.
