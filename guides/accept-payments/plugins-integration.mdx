# Métodos de Pago

Paylands soporta una amplia variedad de métodos de pago para adaptarse a las necesidades de tus clientes.

## Tarjetas de Crédito y Débito
Acepta pagos con las principales marcas de tarjetas:
- Visa
- Mastercard
- American Express
- Y más...

## Pagos Alternativos
- **Bizum**: Permite a tus clientes pagar de forma rápida y segura usando su móvil.
- **Transferencias**: Soporte para transferencias bancarias.

## Otros Servicios
Paylands se integra con múltiples procesadores y gateways para ofrecer la mejor cobertura.
