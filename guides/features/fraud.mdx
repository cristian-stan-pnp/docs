# Fraud Prevention

Paylands implements a set of fraud prevention measures to ensure the security of your transactions.

## Fraud Prevention

Paylands implements a set of fraud prevention measures to ensure the security of your transactions.

- **Email**: Correos de confirmación, notificaciones, etc.
- **SMS**: Mensajes de texto para validaciones o notificaciones.
- **Carta de Pago**: La página web donde el usuario introduce sus datos de tarjeta.
- **Carta de Tokenización**: Interfaz para guardar tarjetas.
- **DCC**: Pantallas de conversión de divisas.

## Variables Predefinidas
Las plantillas soportan variables dinámicas para personalizar el contenido con datos de la transacción (ej. `{{amount}}`, `{{order_uuid}}`).

## Multiidioma
Paylands soporta múltiples idiomas, permitiendo definir plantillas específicas para cada idioma soportado.
