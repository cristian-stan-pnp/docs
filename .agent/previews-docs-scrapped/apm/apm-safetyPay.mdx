En primer lugar, para solicitar un servicio de pago en sandbox para Safetypay deberá ponerse en contacto con **[soporte@paylands.com](mailto:soporte@paylands.com)**.

## Banca por internet[​](#banca-por-internet "Enlace directo al Banca por internet")

* El método de pago safetypay solo esta disponible mediante [**redirección**](/reference/#operation/redirection-payment).
* En la generación de la orden siempre se debe enviar el parámetro "secure": false,
* En la generación de la orden se debe indicar en el extra\_data los siguientes datos obligatorios:

```
"extra_data": {    "profile": {        "first_name": "Nombre Usuario",        "last_name": "Apellido",        "email": "email@example.com"    }}
```

Una vez accedemos al simulador de sandbox de SafetyPay seleccionamos Banca por internet (SP Test Bank Direct). Tras esto iniciamos sesión en la web usando las credenciales admin/admin. Por último, confirmamos la operación.
