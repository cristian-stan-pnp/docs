---

## Panel[​](#panel "Enlace directo al Panel")

Para consultar el saldo disponible en cuenta desde el panel, tenemos que acceder al cliente, detalle y saldo de cuentas. ![Listado de saldos](/images/docs/guides/assets/images/balance-a7447b22537df99f591154a00012e40c.png)

## Api[​](#api "Enlace directo al Api")

Vía api tendremos que lanzar una petición al siguiente endpoint:

```
curl --request GET 'https://api.paylands.com/v1/sandbox/balance/{client_uuid}' \--header 'Authorization: pk_test_3c140607778e1217f56ccb8b50540e00' \--header 'Content-Type: application/json' \
```

Y recibiremos una respuesta como esta:

```
{    "message": "OK",    "code": 200,    "current_time": "2024-04-16T13:10:26+0200",    "balances": [        {            "service_uuid": "F4EB8DF8-6369-4E35-8BD0-45A29AF90616",            "type": "DEVENGO",            "service_name": "Devengo Terminal",            "identifier": "ES2067130002000000458101",            "total_amount": {                "amount": "26205.05",                "currency": "EUR"            },            "available_amount": {                "amount": "26205.05",                "currency": "EUR"            }        }    ]}
```
