## Crear URL de cobro[​](#crear-url-de-cobro "Enlace directo al Crear URL de cobro")

1. Navega en el panel de administración a la sección **Pagos/Cobrador de deudas** y pulsa en **"Nuevo enlace"**.

   ![](/images/docs/guides/img/docs/guides/debt-collector/debt-collector-admin-1.png)

2. Se abrirá un modal donde se solicitará la siguiente información para la creación de un enlace:

   | Campo                                  |                         Descripción                        |
   | -------------------------------------- | :--------------------------------------------------------: |
   | **Checkout de pago**                   |                         Obligatorio                        |
   | **Nombre del enlace**                  |                         Obligatorio                        |
   | **Descripción del enlace**             |                         Obligatorio                        |
   | **Contraseña de acceso al enlace**     |                         Obligatorio                        |
   | **Cantidad total a pagar**             |                         Obligatorio                        |
   | **Permitir pagar un importe menor**    |          Opcional (por defecto no está permitido)          |
   | **Fecha de expiración del enlace**     |            Opcional (por defecto nunca expirará)           |
   | **Asociar a un cliente**               | Opcional (por defecto no estará asociado a ningún cliente) |
   | **URL donde se notificarán los pagos** |           Opcional (por defecto no se notificará)          |

3. Introduce la información y pulsa en **"Confirmar"**.

   ![](/images/docs/guides/img/docs/guides/debt-collector/debt-collector-admin-2.png)

4. El enlace se habrá creado. Ahora puedes copiarlo pulsando sobre los tres puntos.

   ![](/images/docs/guides/img/docs/guides/debt-collector/debt-collector-admin-3.png)

## Experiencia del cliente[​](#experiencia-del-cliente "Enlace directo al Experiencia del cliente")

1. **Acceso al enlace**: Al acceder al enlace, se pedirá la contraseña de acceso, la cual se puede consultar en la tabla de URLs del cobrador de deudas.

   ![](/images/docs/guides/img/docs/guides/debt-collector/debt-collector-1.png)

2. **Visualización de la deuda**: Una vez iniciada la sesión, se podrá ver el importe total de la deuda, la cantidad ya pagada y un historial de las transacciones realizadas sobre dicha deuda.

   ![](/images/docs/guides/img/docs/guides/debt-collector/debt-collector-2.png)

3. **Realizar un pago**: Introduce el importe y pulsa en "Pagar". (El importe solo será modificable si se eligió "permitir pagar un importe menor" al crear la URL).

   ![](/images/docs/guides/img/docs/guides/debt-collector/debt-collector-3.png)

4. **Finalización del pago**: Una vez finalizado el pago, se podrá ver el importe de la deuda ya pagado y el importe restante. Si la deuda aún no ha sido saldada, se podrá volver a acceder a este mismo enlace para pagar la cantidad restante.

   ![](/images/docs/guides/img/docs/guides/debt-collector/debt-collector-4.png)
