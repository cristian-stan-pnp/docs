---
title: "Wallets"
description: "Paylands permite al comercio conectar multiples servicios como pasarelas de pago. Un servicio de pago es una pasarela de pago de un determinado proveedor y que Paylands pone a disposición del comercio para su uso de forma transparente y teniendo resuelta la integración con el mismo. De este modo, el comercio puede configurar las pasarelas de forma integrada sin tener que preocuparse en desarrollos con los distintos proveedores."
---

Disponemos de un amplio catálogo de tipos de servicios de pago que el comercio podrá [**solicitar**](#activaci%C3%B3n-de-servicio-de-pago) según las necesidades. Para poder empezar a vender es necesario configurar al menos una pasarela de pago para un tipo de servicio.

## Tipos de wallet[​](#tipos-de-wallet "Enlace directo al Tipos de wallet")

A continuación se muestra una **tabla comparativa** de los servicios de pago disponibles en Paylands destacando las características de cada uno. Podrás elegir el servicio o servicios de pago que mejor se adapten a las necesidades del comercio en base a las características que ofrece cada uno.

| Wallet                                           | Recurrencias | Devoluciones | Países soportados |
| ------------------------------------------------ | :----------: | :----------: | :---------------: |
| [**GOOGLE PAY**](/docs/wallets/wallet-googlepay) |       X      |       ✓      |       Global      |

## Cuando usar la integración por web o app[​](#cuando-usar-la-integración-por-web-o-app "Enlace directo al Cuando usar la integración por web o app")

* **Integración simple:** carta de pago
  * Sólo acepta WEB
* **Integración avanzada:** el comercio integra directamente Google Pay/Apple Pay
  * Acepta WEB y APP
