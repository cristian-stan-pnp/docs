---
title: "Personalización"
description: "Para personalizar tu checkout, accede a la pestaña"
---

![](/images/docs/checkout/img/docs/checkout/customize/customize-1.png)

En ella podrás ver una lista de todos los elementos que puedes personalizar en tu checkout.

| Formulario de personalización |                                                           |             |
| ----------------------------- | :-------------------------------------------------------: | :---------: |
| Título                        |                    Nombre del checkout                    | Obligatorio |
| Tema                          |              Color de fondo (claro o oscuro)              |   Opcional  |
| Descripción                   |       Texto que se mostrará en el resumen de compra       |   Opcional  |
| Fuente                        |                       Tipo de letra                       |   Opcional  |
| Color principal               |      Color de ciertos elementos como botones o iconos     |   Opcional  |
| Idioma                        |           Idioma en que se mostrará el checkout           |   Opcional  |
| Logo                          |   URL de la imagen que se mostrará en la parte superior   |   Opcional  |
| Logo del footer               |   URL de la imagen que se mostrará en la parte inferior   |   Opcional  |
| Términos y condiciones        |        URL de los términos y condiciones de tu web        |   Opcional  |
| Campos adicionales            | Tabla clave/valor que se mostrará en el resumen de compra |   Opcional  |
| Habilitar botón de cancelar   |    Permite mostrar o no el botón para cancelar el pago    | Obligatorio |

![](/images/docs/checkout/img/docs/checkout/customize/customize-2.png)

Debajo dispondrás de una vista previa de cómo se verá el checkout en función de los elementos que hayas personalizado. Esto es solo para comprobar como se verá el checkout, no podrás realizar pagos reales desde esta vista previa.

![](/images/docs/checkout/img/docs/checkout/customize/customize-3.png)

Pulsa en *Guardar* y habrás personalizado tu checkout. Esta personalización podrá ser modificada en cualquier momento.

![](/images/docs/checkout/img/docs/checkout/customize/customize-4.png)
