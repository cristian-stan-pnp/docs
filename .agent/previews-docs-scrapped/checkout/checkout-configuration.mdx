---
title: "Configuración"
description: "Para configurar métodos de pago en tu checkout, accede a los detalles de tu checkout y pulsa en la pestaña"
---

![](/images/docs/checkout/img/docs/checkout/configure/configure-1.png)

En ella podrás ver una lista de todos los métodos de pago disponibles en función de los servicios que tengas habilitados. Si necesitas habilitar un método de pago para el cual no tienes un servicio disponible, deberás contactar con *[soporte@paylands.com](mailto:soporte@paylands.com)* solicitándolo.

![](/images/docs/checkout/img/docs/checkout/configure/configure-2.png)

> **Nota:** Algunos métodos de pago como Google Pay, Apple Pay o ClickToPay no se pueden configurar desde esta sección, ya que estos métodos de pago requieren de una configuración asociada a tu propio comercio en Paylands. Ve a la sección de estos métodos de pago para saber como configurarlos.
>
> * [Google Pay](/docs/wallets/wallet-googlepay)
> * [Apple Pay](/docs/wallets/wallet-applepay)
> * [ClickToPay](/docs/wallets/mastercard-clicktopay)

Al marcar como visible un método de pago, se asignará un servicio automáticamente, aunque podrás modificarlo en caso de disponer de varios servicios para este método de pago.

![](/images/docs/checkout/img/docs/checkout/configure/configure-3.png)

Debajo dispondrás de una vista previa de cómo se verá el checkout en función de los métodos de pago que tengas habilitados. Esto es solo para comprobar como se verá el checkout, no podrás realizar pagos reales desde esta vista previa.

![](/images/docs/checkout/img/docs/checkout/configure/configure-4.png)

Pulsa en *Guardar* y habrás configurado tu checkout. Esta configuración podrá ser modificada en cualquier momento.

![](/images/docs/checkout/img/docs/checkout/configure/configure-5.png)
