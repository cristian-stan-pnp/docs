---
title: "Credit fund transfer (CFT)"
description: "Paylands permite transferir dinero del comercio al usuario mediante un abono a tarjeta, también conocido como Credit Fund Transfer o CFT. Esta modalidad de pago está disponible tanto para tarjetas Visa como Mastercard."
---

En primer lugar, es necesario [**generar la orden de pago**](/reference/#operation/generate-payment-order). Es importante que la operativa sea `PAYOUT` y setear el campo `secure: false`. El resto de datos variarán dependiendo de tu integración. Además de los campos habituales, **hay que indicar algunos datos adicionales del usuario**, que son los siguientes: nombre, apellidos, email y número de teléfono del usuario.

Ejemplo de generación de la orden de pago (POST /payment)

```
{    "signature": "341f7de8e6fc49da8d8736473af6b03a",    "amount": 1245,    "operative": "PAYOUT",    "secure": false,    "customer_ext_id": "test",    "service": "9A1BDCC8-DB30-4ED2-8523-62B330A67873",    "description": "Ejemplo CFT",    "url_post": "https://tucomercio.com/procesarpago",    "extra_data": {        "profile": {            "first_name": "Nombre",            "last_name": "Apellidos",            "email": "email@gmail.com",            "phone": {                "number": "600999999",                "prefix": "34"            }        }    }}
```

En segundo lugar hay que realizar el [**abono a tarjeta**](/reference/#operation/payout-payment) indicando el UUID de orden que acabamos de generar y el UUID de la tarjeta del usuario. Se recomienda que para evitar posibles rechazos la tarjeta haya tenido al menos un pago exitoso.

Ejemplo de la petición de abono a tarjeta (POST /payment/payout)

```
{    "signature": "121149a0ba5361191d740fa898784a8b",    "order_uuid": "1F405EA3-9798-42A6-9E87-BD347EF67F55",    "source_uuid": "C10721E7-1404-45DC-8762-351DD9945D1D",    "customer_ip": "127.0.0.1"}
```

Hecho esto, inmediatamente recibiremos la respuesta con el resultado del abono, que puede resultar aceptado o rechazado. Adicionalmente, se enviará la notificación del pago a la `url_post` indicada al generar la orden de pago.

**NOTA:** La transferencia de fondos es final, no es posible revertir un abono a tarjeta. Los fondos pueden tardar algunos días en llegar al destinatario, dependiendo de su banco.
