En la sección **Herramientas de Pago →** [**Enlaces de pago**](https://backend.paylands.com/payment-urls) de nuestro Backend dispones de una potente herramienta para crear enlaces de pago y hacérselos llegar a tus clientes de forma muy sencilla. En ella aparece el listado de enlaces creados. Para crear un nuevo enlace deberás:

### 1. Crear enlace[​](#1-crear-enlace "Enlace directo al 1. Crear enlace")

Pulsando el botón *"Nuevo Enlace"* se abrirá el formulario para crear un nuevo enlace.

![Link payments form](/images/docs/integration/assets/images/link-payments1-927698ed2ffe3d563cc1bc2b991f0837.jpg)

Deberás rellenar los siguientes campos y pulsar enviar:

* **Descripción.** Una pequeña descripción de para que/quién has creado este enlace.
* **Importe por defecto.** El importe que deseas cobrar al cliente. La cantidad se debe introducir en céntimos. Ej: para cobrar 50€ se debe introducir 5000 como valor.
* **Servicio de pago.** Selecciona el servicio de pago que tengas configurado.
* **Tipo de pago.** Selecciona el tipo de pago sobre el cual quieres hacer la transacción: NO SEGURO, 3DS, BIZUM.
* **Plantilla.** Puedes seleccionar una plantilla donde hayas personalizado la carta de pago que se mostrará al cliente. Si no seleccionas ninguna se enviará el diseño de carta de pago por defecto.
* **Idioma.** Puedes seleccionar el idioma de la carta de pago. Si no seleccionas ninguno se enviará en el idioma por defecto que tengas configurado.

<Info>
  Puedes personalizar la apariencia de la carta de pago en **Personalización → [Plantillas](https://backend.paylands.com/templates)**, creando una nueva plantilla de tipo *Carta de Pago*.
</Info>

### 2. Copiar enlace y compartir[​](#2-copiar-enlace-y-compartir "Enlace directo al 2. Copiar enlace y compartir")

Una vez enviado el formulario, aparecerá en el listado el enlace recién creado. Pulsando en el icono \[+] se desplegará una sección donde podrás acceder al enlace de pago.

![Link payments copy](/images/docs/integration/assets/images/link-payments2-08a5cd9744eaef41d0d3416c690a9dd1.jpg)

Pulsa sobre el icono de enlace y se abrirá la carta de pago. Puedes copiar dicho enlace y hacérselo llegar al cliente para que realice el pago.

![Payment Card](/images/docs/integration/img/link_payments3V2.png)
