---
title: "Pagos Mo/To"
description: "Paylands ofrece una funcionalidad llamada \"Pagos Mo/To\" para procesar pagos offline. Mo/To es el acrónimo para Mail Order / Telephone Order. Este método permite realizar el pago de una orden con los datos de tarjeta obtenidos mediante email o teléfono."
---

Esta operación no requiere que el usuario lleve a cabo el proceso de autenticación 3D Secure ya que está exenta de cumplir la normativa PSD2. En caso de tener activado el antifraude, las órdenes consideradas de riesgo se procesarán sin ninguna restricción adicional.

### Pagos Mo/To tokenizados[​](#pagos-moto-tokenizados "Enlace directo al Pagos Mo/To tokenizados")

Si se ha tokenizado la tarjeta con anterioridad se puede indicar el SOURCE\_UUID de la tarjeta y el CUSTOMER\_EXT\_ID al cual se asoció la tarjeta para que el pago se lleve a cabo directamente a través del 3DS, sin que el usuario tenga que indicar los datos de la tarjeta en la carta de pago. Los pagos tokenizados son SEGUROS independientemente de si se indica el campo `secure` a TRUE o no.
