---
title: "Introducción"
description: "El PROXY PCI de PaynoPain facilita la captura y transmisión de los datos de las tarjetas de pago sujetas a la normativa PCI."
---

<Info>
  Con la nueva normativa europea PSD2 es más importante que nunca que su software y aplicaciones no almacenen los datos de las tarjetas directamente, sino que los deleguen en un socio que cumpla la normativa PCI, como PaynoPain.
</Info>

El PROXY PCI sirve para lo siguiente:

* Actuar como proxy entre su backend y una aplicación de terceros, como un motor de reservas o un gestor de canales. El PROXY PCI almacenará los datos de la tarjeta PCI por usted en un sistema seguro de bóveda que cumple con la norma PCI y asignará a estos datos un alias que se le remitirá a usted y mantendrá todos los demás datos tal como están. Estos datos de tarjeta enmascarados también pueden ser reenviados a otro software backend como una pasarela de pago, un banco, etc. Todo esto se puede conseguir utilizando Canales.
* Permite capturar los datos de la tarjeta directamente en el lado del cliente utilizando iframes seguros para que los datos sean capturados directamente en los servidores de PCI PROXY. Esto se puede integrar fácilmente utilizando la librería javascript.
* Convertirse en una herramienta para mostrar los datos de la tarjeta a uno de los empleados de su empresa utilizando un token de visualización y la interfaz web de PCI PROXY.
* Permite utilizar los datos capturados para autorizar/verificar los datos de la tarjeta o incluso procesar un pago con la pasarela de procesamiento de pagos incorporada [Paylands](http://www.paylands.com/).

<Info>
  El PROXY PCI viene con el procesamiento de pagos incorporado impulsado por [Paylands](http://www.paylands.com/)
</Info>

Para poder utilizar las diferentes funcionalidades necesitará integrar la API REST de PROXY PCI. Esta guía hará referencia a los diferentes métodos disponibles en esta API REST. Puede acceder a la documentación detallada de esta API [aquí](https://pci-proxy-api.paynopain.com/docs) y para poder utilizarla debe solicitar primero sus credenciales de cliente enviándonos un correo electrónico a [soporte@paylands.com](mailto:soporte@paylands.com).
