</Info>

## Cum funcționează?

Pentru a lucra cu abordarea de tokenizare a clientului, trebuie să urmați pașii de mai jos:

1. Primul lucru pe care trebuie să-l faceți este să generați un `token` de captură utilizând [POST /tokenize/\{reference}](https://pci-proxy-api.paynopain.com/docs#/Client%20tokenization/IssueTokenizationToken) în API-ul REST. În timpul generării acestui token trebuie să furnizați o `referință`. Această `referință` trebuie să fie unică și poate fi id-ul rezervării dumneavoastră, booking-ului, etc.
2. Odată ce aveți token-ul, trebuie să inițializați biblioteca javascript cu acest token. Mai multe detalii despre acest lucru mai jos.
3. Biblioteca javascript se inițializează și afișează un câmp de intrare iframe securizat pe site-ul dumneavoastră.
4. Utilizatorul introduce datele sale în formularul web, iar numărul pan este introdus în câmpul iframe securizat.
5. Utilizatorul apasă pe trimitere, iar dumneavoastră îi spuneți bibliotecii să trimită datele către motorul nostru de seif.
6. Apoi puteți obține rezultatul pe serverul dumneavoastră inițiind un apel către metoda [GET /card/reference/\{reference}](https://pci-proxy-api.paynopain.com/docs#/Client%20tokenization/GetTokenizationResults). Rezultatul va include pan-ul tokenizat al cardului printre alte detalii.
7. Acest token poate fi utilizat pentru a trimite datele cardului către un terț sau pentru a emite un token de vizualizare.

<Warning>
  API-ul REST trebuie apelat întotdeauna direct de pe serverul dumneavoastră, niciodată din partea clientului.
</Warning>

## Includerea bibliotecii javascript

Vă rugăm să încărcați versiunea corectă a bibliotecii javascript direct de pe serverele noastre compatibile PCI, în funcție de mediu.

SANDBOX

```
<script src="https://pci-proxy-api.paynopain.com/sandbox/client.js"></script>
```

PRODUCTION

```
<script src="https://pci-proxy-api.paynopain.com/prod/client.js"></script>
```

### Inițializarea bibliotecii

Pentru a inițializa biblioteca trebuie să furnizați id-ul elementului DOM unde biblioteca trebuie să injecteze iframe-ul cu câmpul de intrare pan împreună cu `token`-ul emis cu API-ul REST. Odată ce iframe-ul a terminat de încărcat, un eveniment `ready` va fi declanșat.

```html
<div id="pan"></div>
<button id="submit">Enviar</button>
```

```

```js
var proxyFields = new ProxyFields();
proxyFields.init('::token::', { cardNumber: "pan" });
proxyFields.on('ready', () => {
    console.log('iframe ha terminado de cargarse');
});
```

### Stilizarea câmpului de intrare

Metoda recomandată pentru a stiliza câmpul de intrare este utilizarea metodei `setStyle`. Aceasta trebuie făcută în callback-ul `ready`.

```js
proxyFields.on("ready", function() {
  proxyFields.setStyle(
    "cardNumber",
    "width: 300px; border-radius: 4px; border: 1px solid black; padding:5px 10px;font-size:100%;"
  );
  proxyFields.setStyle("cardNumber::placeholder", "color: grey");
  proxyFields.setPlaceholder("cardNumber", "4987806170805591");
});
```

### Validarea și trimiterea datelor

Odată ce utilizatorul a completat formularul de trimitere, puteți apela metoda `submit`. Metoda `submit` primește ca unic parametru un obiect în care trebuie să furnizați informații suplimentare despre card. Aceste date pot fi colectate direct de către dumneavoastră, deoarece nu sunt supuse normativei PCI.

Odată ce ați apelat metoda `submit`, un callback `validate` va fi declanșat cu un obiect de intrare care conține informații legate de eroare, dacă există. Dacă pan-ul este valid, trimiterea va avea loc și un callback de succes sau eroare va fi declanșat. După ce callback-ul `success` este declanșat, puteți apela în siguranță [metoda API-ului REST](https://pci-proxy-api.paynopain.com/docs#/Client%20tokenization/GetTokenizationResults) pe serverul dumneavoastră pentru a obține token-ul pan.

```js
proxyFields.submit({
  expiryMonth: "01",
  expiryYear: "21",
  cardHolder: "Name Displayed On Card"
});
proxyFields.on("validate", function(data) {
  if(validate.hasErrors){
    console.log("Card pan is not valid");
    proxyFields.setStyle("cardNumber","border: 1px solid red");
  }
});
proxyFields.on("success", function(data) {
  console.log('Submission succeeded');
});
proxyFields.on("error", function(error) {
  console.log('Ha ocurrido algo malo');
});
```
