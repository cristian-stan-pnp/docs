## Introdução

Floa é uma solução de pagamento francesa que permite financiar compras com opções de crédito ao consumidor, incluindo pagamentos parcelados. Entre as opções de parcelamento estão:

* 3X: Pagamentos em 3 parcelas, com 30 dias de diferença entre cada uma.
* 4X: Pagamentos em 4 parcelas, com 30 dias de diferença entre cada uma.

## Dados extras obrigatórios

Dependendo do tipo de produto ou serviço oferecido, será necessário indicar informações adicionais dentro do `extra_data` durante a criação da ordem, embora o país seja sempre obrigatório (`extra_data.billing_address.country`), juntamente com o nome e sobrenome (`extra_data.profile.first_name` e `extra_data.profile.last_name`, respectivamente).

Dependendo do produto, existem os seguintes setores de atividade: **General**, **Stay** (hotéis) e **Travel**.

## Métodos de pagamento em Sandbox

É recomendável usar um valor entre 100€ e 250€ para aumentar as probabilidades de elegibilidade bem-sucedida.

Os dados do cartão de teste são os seguintes:

```
- número do cartão OK: 5017 6791 1038 0400
- número do cartão KO: 5017 6791 0090 0712
- CVV: 000
- data de validade: posterior à data da última prestação
```

## Moeda

A moeda suportada para este serviço é o euro (EUR).

## Códigos de erro

A estrutura do erro tem o seguinte formato:

```
Código - Mensagem
```

Por exemplo:

```
3 - Failed (Technical failure)
```
