To activate it, you must access the Paylands panel and navigate to the sidebar menu Administration > Clients. Once there, access the details of the client for which you want to activate this option and go to the configuration preferences tab.

![Clients Preferences](/images/docs/integration/assets/images/configuration-clients-preferences-b214cf1aa1dc9fb553b91dbcc08cd8e8.png)

Once here, you can activate **Generate payment URLs** and if you want a shortened version of the URL, you can activate **Generate shortened URLs**. It is important to note that the first option must be activated if you want to receive shortened URLs.

## URL generation:

These three URLs are generated automatically, but in the case of `bizum` and `3ds_tokenized`, certain requirements must be met.

**Bizum:** This URL will only be generated if the service specified when creating the order supports this payment method. [**Bizum**](/docs/apm/apm-bizum)

**3ds\_tokenized:** To generate this URL, you need to specify `secure: true` and a valid `source_uuid` when creating the order. [**3Ds Tokenized**](/reference/#operation/3ds-tokenized)
