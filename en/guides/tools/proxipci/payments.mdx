---
title: 'Payments with PROXY PCI'
description: 'How to process payments through the Paylands PROXY PCI'
---

To charge a saved card, you first need to generate a payment order. This is done using [the order generation method](https://docs.paylands.com/reference/#operation/generate-payment-order). You need to use the card token obtained during the tokenization process and send it as the value of the `source_uuid` field. After this, you can [redirect the user to a payment form](https://docs.paylands.com/reference/#operation/redirection-payment) or [use the web service directly](https://docs.paylands.com/reference/#operation/webservice-payment).

<Info>
  Please refer to the complete PAYMENT GATEWAY documentation: [https://docs.paylands.com](https://docs.paylands.com)
</Info>
