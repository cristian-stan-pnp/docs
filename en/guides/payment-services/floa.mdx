## Introduction

Floa is a French payment solution that allows financing purchases with consumer credit options, including installment payments. The installment payment options offered are:

* 3X: Payments in 3 installments, with 30 days between each one.
* 4X: Payments in 4 installments, with 30 days between each one.

## Required extra data

Depending on the type of product or service offered, it will be necessary to provide additional information within `extra_data` during order creation, although the country will always be required (`extra_data.billing_address.country`), along with the first name and last name (`extra_data.profile.first_name` and `extra_data.profile.last_name`, respectively).

The following table shows the additional fields that must be sent within `extra_data.payment` depending on the type of product offered:

| Field                    | Description                                                                                                                                                                                                                                                                                                                                                                                            | Type       | Required    |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------- | ----------- |
| categorie\_1             | Category of the most expensive product. For Travel/Stay use one of the following values: **AIRLINE TICKET**, **TRAIN TICKET**, **BUS TICKET**, **HOTEL**, **CAMPING**, **STAY**, **CRUISE**, **CAR RENTAL**                                                                                                                                                                                            | string(30) | Yes         |
| categorie\_2             | Subcategory of the most expensive product. Optional for Travel.                                                                                                                                                                                                                                                                                                                                        | string(30) | Yes         |
| cart\_item\_count        | Number of items                                                                                                                                                                                                                                                                                                                                                                                        | int        | Yes         |
| sale\_channel            | Sales channel. One of the following values: **DESKTOP**, **TABLET**, **TABLET\_IPAD**, **SMARTPHONE**, **SMARTPHONE\_ANDROID**, **SMARTPHONE\_IPHONE**                                                                                                                                                                                                                                                 | string(18) | Yes         |
| merchant\_home\_url      | Merchant home page URL                                                                                                                                                                                                                                                                                                                                                                                 | string     | Yes         |
| merchant\_back\_url      | URL to return to the previous page of the merchant's site                                                                                                                                                                                                                                                                                                                                              | string     | Yes         |
| departure\_date          | Start date format: DD/MM/YYYY HH:MM                                                                                                                                                                                                                                                                                                                                                                   | date       | Travel/Stay |
| return\_date             | Return date format: DD/MM/YYYY HH:MM                                                                                                                                                                                                                                                                                                                                                                  | date       | Travel/Stay |
| traveller\_count         | Number of travelers                                                                                                                                                                                                                                                                                                                                                                                    | int        | Travel/Stay |
| own\_ticket              | Field to identify whether the credit card holder used to finance the trip is participating.                                                                                                                                                                                                                                                                                                            | bool       | Travel/Stay |
| room\_range              | For Stay payments, the hotel range (stars)                                                                                                                                                                                                                                                                                                                                                             | string     | Stay        |
| insurance                | Whether the customer chooses insurance. Values: "yes", "no"                                                                                                                                                                                                                                                                                                                                            | string     | Travel/Stay |
| destination\_country     | Destination country code, in ISO 3166-2 format (e.g.: "ES" for Spain)                                                                                                                                                                                                                                                                                                                                 | string     | Travel/Stay |
| travel\_type             | A value from: **Unknown**, **OneWay**, **TwoWay** (round trip), **Multiple** (multiple destinations)                                                                                                                                                                                                                                                                                                   | string(2)  | Travel/Stay |
| departure\_airport       | IATA code of the departure airport                                                                                                                                                                                                                                                                                                                                                                     | string(3)  | Travel      |
| arrival\_airport         | IATA code of the destination airport                                                                                                                                                                                                                                                                                                                                                                   | string(3)  | Travel      |
| main\_departure\_company | For air travel, name of the airline for the first flight (IATA code)                                                                                                                                                                                                                                                                                                                                   | string(3)  | Travel      |
| shipping\_method         | Merchant delivery method. If not sent, "VIR" by default. Values: **STD** (Standard), **EXP** (Express Delivery), **TRK** (Tracked), **REG** (Registered), **COL** (Colissimo), **CHR** (Chronopost), **REL** (Mondial Relay), **TNT** (TNT), **UPS** (UPS), **TRP** (Transporter), **MAG** (Delivery in store), **LCK** (Lockers), **VIR** (Virtual), **RIM** (Immediate withdrawal in store)          | string(3)  | No          |
| fees\_amount             | Total amount of fees associated with the order (in cents)                                                                                                                                                                                                                                                                                                                                              | string     | No          |
| shipping\_cost           | Total amount of shipping costs for the order (in cents)                                                                                                                                                                                                                                                                                                                                                | string     | No          |
| discount\_amount         | Total amount of discounts associated with the order (in cents)                                                                                                                                                                                                                                                                                                                                         | string     | No          |

Depending on the product, the following business sectors exist: **General**, **Stay** (hotels), and **Travel**.

Below is an example of `extra_data` sent to make a payment for each business sector:

```

```

## Sandbox payment methods

It is recommended to use an amount between €100 and €250 to increase the chances of successful eligibility.

The test card details are as follows:

```
- card number OK: 5017 6791 1038 0400
- card number KO: 5017 6791 0090 0712
- CVV: 000
- expiration date: later than the last installment date
```

## Currency

The supported currency for this service is the Euro (EUR).

## Error codes

The error structure has the following format:

```
Code - Message
```

For example:

```
3 - Failed (Technical failure)
```
